This plugin resolved issue of align images in backend editor for old themes.

== Description ==

In latest version of wordpress the wide and full width align image functionality not working in older themes at backend post editor.This plugin resolved this issue.

Just Activate the plugin and you are ready to go.
 
== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
=is wide image align not worked in twentysixteen ?=
NO, image align feature of latest version of wordpress only working with latest theme twentynineteen.

=How can i use this feature in old theme version in wordpress?=
Using this plugin you can access this feature in old themes like twentysixteen,twentyseventeen and twentyfifteen etc..
